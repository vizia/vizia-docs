Struct vizia::entity::Entity

source · [−]
pub struct Entity(_);
Expand description

An entity is an identifier used to reference a view; to get/set properties in the context.

Rather than having widgets own their data, all state is stored in a single database and is stored and loaded using entities.

This type is part of the prelude.

impl Entity

source

pub fn root() -> Entity

Creates a new root entity.

The root entity represents the main window and is always valid. It can be used to set properties on the primary window, such as background color, as well as sending events to the window such as Restyle and Redraw events.
